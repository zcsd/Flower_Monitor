# Edge Sensor (ESP32-MQTT-WIFI)

## Hardware:
- ESP-32S Wifi support

- DHT11/22 Temperature and Humidity sensor, connect with ESP-32S.
- HC-SR501 Motion Sensor, connect with ESP-32S.
